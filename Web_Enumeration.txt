++ Escaneo manual
--- Probar acceso a directorios de un servidor web, ejemplo: <url_servidor>/<nombre_carpeta>
--- Acceder a las developer tools de firefox para examinar el contenido del sitio web. Acceso mediante: Ctrl + Shift + I 
----- Nota: el acceso también es posible mediante la barra superior en Tools ---> Browser tools ---> Web Developer Tools
----- Nota: En la sección headers, parte inferior derecha podemos ver información relacionada al tipo de servidor. ESta sección se habilita al ubicar algún directorio, por ejemplo / 

++ Escaneo automatizado (usando worldlists y gobuster)
--- Ruta de las worldlists por defecto: usr/share/worldlists
--- Instalar worldlists adicionales: sudo apt-get install seclists
--- gobuster (para ejecutarlo en terminal o instalarlo si no está instalado)
A continuación algunos ejemplos de comandos con gobuster:
----- gobuster dir -u <url_sitio_web> -w /usr/share/worldlists/dirbuster/<nombre_worldlist> (Usa un diccionario para verificar la estructura de un sitio web)
----- Tras el escaneo de haber coincidencias con el worldlist aparecerán los 
directorios asociados
----- gobuster dir -u <url_sitio_web/<directorio>> -w </usr/share/worldlists/dirbuster/<nombre_worldlist> -x.conf,.txt,.pdf,.doc,.html,.docx,.xls,.mp3,.mp4,.js -k
(Usa un diccionario para verificar la presencia de archivos en la estructura de un sitio web)
----- gobuster dns -d <url_sitio_web> -w </usr/share/worldlists/dirb/<nombre_worldlist> (Permite escanear los subdominios asociados al sitio web)


++ Esaneo de sitios webs basados en Wordpress (WPS Scan)
--- wpscan
--- Si no está instalado, hacerlo con: sudo apt-get install wpscan
--- wpscan --update (Actualiza la base de datos de wpscan)
--- wpscan --url <url> --enumerate t (Realiza el escaneo a un sitio web basado en wordpress)
--- wpscan --url <url> --enumerate p (Realiza el escaneo a un sitio web basado en wordpress, en especifico busca información relacionada con los plugins)
--- wpscan --url <url> --enumerate u (Realiza el escaneo a un sitio web basado en wordpress, en especifico busca información relacionada con los usuarios)
--- wpscan --url <url> --enumerate v (Realiza el escaneo a un sitio web basado en wordpress, en especifico busca información relacionada con las vulnerabilidades, 
la v puede ir acompañada de t p ó u)
--- wpscan --url <url> --passwords <worldlist> --username <usuario> (Realiza un ataque basado en diccionario para encontrar la contraseña de un usuario asociado
al url de wordpress)
