++ PC1: En estos momentos se ha establecido el reverse shell, lo que nos permite
ejecutar comandos, tales como:
---- cat /etc/os-release (Permite ver la version de distribución GNU/Linux instalada)
---- hostname (Permite ver el nombre del equipo)
---- cat /etc/group (Permite ver información referente a los grupos)
---- cat /etc/passwd (Permite ver el listado de los usuarios del sistema)
---- sudo cat /etc/shadow (Permite ver los hashes de las contraseñas de los usuarios del sistema. Nota: el hash es aquel que se encuentra entre los :)
---- cd /var/mail/ (De ser un servidor web permite ver información relacionada al 
sistema y los usuarios)
---- dpkg -l (En distribuciones basadas en Debian permite ver el listado de los paquetes instalados)
